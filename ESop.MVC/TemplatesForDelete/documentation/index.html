نحوه بروزرسانی:

همیشه می توانید بروزرسانی جدید این قالب را به صورت رایگان از قسمت پنل کاربری>دانلودها، دانلود کنید.

از شما خواسته می شود پوشه ها و فایل ها را بروز کنید و به ویژه مطمئن شوید که پوشه theme-assets نیز بروز شده است. گاهی اوقات، ساختار HTML نیز نیاز به بروزرسانی دارد. بنابراین، اگر برخی از عناصر ویرایش شده شما کار نمی کند، ساختار آن را با ساختار قالب ما (آخرین نسخه) مقایسه کنید و تغییرات مورد نیاز را اعمال کنید.

نکته:

برای هر گونه شخصی سازی قالب، حتما طبق گفته های قسمت (نحوه شخصی سازی) عمل کنید تا بعد از بروزرسانی تغییرات شما از بین نرود.

نحوه شخصی سازی:

اگر می خواهید قالب را شخصی سازی کنید ، توصیه می شود کد CSS و JS خود را به عنوان فایل های جدا با استفاده از تگ های link و script به پروژه اضافه کنید. نام فایل CSS پیشنهادی theme-asset/css/custom.css و نام فایل JS پیشنهادی theme-asset/js/custom.js است.

اطمینان حاصل کنید که فایل CSS خود را در انتهای لیست استایلها اضافه کردید یا فایل JS را در انتهای لیست اسکریپت ها.

نکته:

هر دوی این فایلها برای راحتی شما در قالب وجود دارد و در صفحات لینک شده است.

نکته:

تحت هیچ شرایطی فایلهای اصلی قالب را شخصی سازی نکنید،زیرا با بروزرسانی قالب کدهای شما از بین خواهد رفت!

ساختار:

  • HTML - main theme folder
    • theme-assets - assets folder
      • css - all common styles used on the pages
      • fonts - all common fonts used on the pages
      • images - images for the project
      • js - all common scripts used on the pages
      • sass - all common styles used on the pages
    • ... .html - Theme Preview

وابستگی ها:

قالب ... با آخرین نسخه فریمورک محبوب Bootstrap طراحی شده است، و به این معنی است که می توانید از قابلیت ها،کامپوننت ها و کلاس های آماده این فریمورک در قالب بهره ببرید.

لازم به ذکر است که در آپدیت های آینده تمامی کامپوننت های این فریمورک با قالب از لحاظ ظاهر یا بصری همخوانی بهتری پیدا خواهد کرد.

از طریق لینک پایین می توانید مستندات فریمورک Bootstrap را بخوانید.

https://getbootstrap.com/
